# Movie Watchlist App

This project is designed for users to search for movies, and add, update, or remove them from their watchlists. This was built to explore how to build fullstack projects.

The backend consists of Node.js, Express, Prisma ORM, and PostregSQL. Additionally, Neon was used to host the database, Requestly was used to test the backend, and JWT was used for user authentication.

The front end is currently not worked on yet, but the plan is to use React, CSS, and HTML as the core components, may explore other tools depending on what I find works best for this project.
